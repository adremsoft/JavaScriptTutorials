## Dependencies

* Node.js v8.1

## Installation

`npm install`

`set DEBUG=*`

### 1. Primitives

`npm run primitives`

### 2. Functions

`npm run functions`

### 3. Functional processing

`npm run functionalProcessing`

### 4. Async tasks

`npm run asyncTasks`

### 5. Prototypes

`npm run prototypes`

### 6. Angular JS

`npm run webServer`

The application will be served at: http://localhost:4000/
